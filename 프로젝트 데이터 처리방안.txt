1. 자소서를 통으로 보고
한줄단위 라벨링
단어를 나눠서
해당하는 X(단어중심), y(인재상)값 학습
모델 작성
-> 자소서 등록
자소서별 인재상 확인

=> 기업 각각 1개의 인재상 소지
-> 기업별로 자소서(여러개의 인재상 소지)
-> 기업별 크롤링한 내역이 이상치로 나올 수 있음

2. 자소서 굿부분으로 등록된 자소서 인재상 파악
자소서 배드부분으로 자소서의 부족한 부분파악(딥러닝부분인지 확인)
+ 기업선택 후 자소서 등록으로 해당기업의 중점가치로 기준삼기

3. 기업별 인재상 모두 살리기
-> 재설정한 인재상에 맞춰서 각각 재정립


데이터 전처리
우선 '다.' 단위 나누기 => 문장 단위 구분
문장단위 인재상 라벨링 => 우리가 설정한 11개 단어로 통일
ex) 열정으로 라벨링시 열정과 비슷한 단어
-> 다른 문장에서 열정과 비슷한 단어가 나올 수 있음
-> 빈도가 높을 때
